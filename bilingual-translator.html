システムの使い方

このアプリは、超低遅延のリアルタイム日⇄英相互翻訳と、AIアシスタント機能を統合したプラットフォームです。

  • リアルタイム翻訳: 中央ペイン。日本語/英語を自動識別してテキスト出力します。
  • AI要約・Q&A: 右ペイン。選択した最新モデルを使い、会話の要約や、文脈に応じた質問に答えます。文字起こしを含めず既存知識のみで答えることも可能です。

接続の手順：

  1. 有効な Gemini APIキー を入力。
  2. 「翻訳セッションを開始」をクリックし、マイクを許可して話しかけます。

マイク利用のためのローカルサーバー起動ガイド

ブラウザの仕様により、HTMLファイルをただダブルクリックした状態 (file://) では、マイクへのアクセスが完全にブロックされます。

安全な通信コンテキストを偽装するため、ご自身のパソコン上で簡易的な「ローカルWebサーバー」を立ち上げて実行する必要があります。

1 Python コマンドを使う場合

  1. ターミナル（Mac）またはコマンドプロンプト（Win）を起動。
  2. このファイルが存在するフォルダへ移動し、次を実行：
python -m http.server 8000

起動後、ブラウザで http://localhost:8000 にアクセスしてください。

※サーバー上で実行すると、マイク許可ダイアログが出現します
